# 2025 FIC 决赛

- **来源**：https://www.cnblogs.com/WXjzc/p/18901128 （WXjzc 博客）
- **检材构成**：5 个 — 手机(检材1, 含Linux容器) + 计算机(检材2, 含加密系统/Linux系统) + 服务器(检材3, 含 WEB-API 容器/SO逆向) + 物联网摄像头(检材4) + APK(检材5, 智能家居)

## 高频套路（自总结）

1. **手机里跑 Linux 容器**是 2025 新趋势 — Termux/Linux Deploy/proot-distro。要先识别容器路径，再当作 Linux 主机分析
2. **SO 逆向 + AES** — 给 SO 文件，找 getAeskey + encrypt 函数，自己写 Python 解密
3. **加密容器套娃** — VeraCrypt 容器里放欠条，欠条里有助记词，助记词推钱包
4. **远程调证 ID** — APK 抓包获取调证 ID，回到平台输入进入下一关（比赛流程设计）
5. **综合多选题** — 必须把前面所有题答完才能反推
