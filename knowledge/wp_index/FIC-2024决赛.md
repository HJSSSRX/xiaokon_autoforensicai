# 2024 FIC 决赛

- **来源**：https://mei-you-qian.github.io/2026/01/22/2024FIC%E5%86%B3%E8%B5%9B/ （玫幽倩）
- **检材构成**：3 个 E01 — data.E01(数据分区) + sys.E01(系统分区) + PersonalPC.E01(个人计算机) + PVE 虚拟化平台 + 软路由 + 云手机集群 + 服务器集群

## 高频套路（自总结）

1. **PVE 是 FIC 大头**：每年都会考。装 Proxmox VE 模拟环境。重点：版本号 / vm 列表 / vmbr 网桥 / smbios / 用户权限 / 快照时间 / shell 历史
2. **PVE → 云手机集群**：node1/node2 是 PVE 上的虚拟机，里面跑 Docker 容器手机（redroid 之类）；要先解锁 PVE 才能看到容器
3. **AI 绘图取证**：识别 SD WebUI / ComfyUI；模型 hash + prompt + seed 三件套
4. **图片隐写套娃**：jpg/png 末尾藏 zip（`binwalk`/`foremost`/`zsteg`），zip 有密码
5. **网站重构**：dump mysql + 部署框架（多为 ThinkPHP / SpringBoot），跑起来才能查后台数据
6. **OpenClash 配置**：`/etc/openclash/config.yaml` 直接看；订阅地址常是关键题
