## 高频套路（自总结）

1. **盘古石的远控工具**：基本是 RustDesk fork，看 `~/.config/rustdesk/RustDesk.toml` 找 relay/key
2. **macOS 逆向**：用 Hopper / Ghidra；Mach-O 二进制；常考自定义加密 + sleep 反调试
3. **智能冰箱**：先看 `/data/data` 找 app db；图片可能藏在 `/sdcard/DCIM/Fridge/`；隐写用 `zsteg`/`stegsolve`
4. **TG 加密聊天**：Telegram secret chat 不在云端备份；要拿设备本地的 `tdata` 或 iOS keychain
5. **monitor app 套路**：Audio threshold + Timer + 黑色 Curtain Activity 是常见监听代码模式
6. **苹果备份爆破**：用 `iTunesBackup` 抓 `Manifest.plist`，扔 `passwarekit` 爆破
