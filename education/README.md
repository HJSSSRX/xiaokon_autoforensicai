# Education Module

Learning paths and student tracking for ForensicAI's education mode.

## Status

Education module is in design phase. See `share/SYSTEM_DESIGN.md` §6 for the full design.

## Planned Contents

```
education/
├─ README.md                    # This file
├─ learning_paths.md            # Auto-generated learning routes
├─ student_progress.yaml        # Per-student progress (gitignored)
└─ assessments/                 # Quiz/test questions
```

## Teaching Levels

- **L1 Guided**: Step-by-step Socratic questioning for beginners
- **L2 Hinted**: Direction hints only, student operates independently
- **L3 Independent**: Only provide question, review after completion
