## 踩坑

1. **E01 内 BitLocker 分区不可直接挂**：需要先 `losetup` 单独切出分区，再试 `ntfs-3g`。本例 E 盘 BitLocker 完全阻塞。
2. **loop 设备 overlap 误报**：mount 失败"overlapping loop"是因为 kernel 把 loop0 默认认为覆盖整个文件，手动 `losetup` 指定 offset/sizelimit 可绕开。
3. **BitLocker 密钥 `.TXT` 才 550 字节**，完全可驻留 MFT record 内，但被 Eraser 擦过后 MFT 也清零。
4. **微软拼音 UDP 字典文件太多空的**：实际数据只在 268 字节的 UDP*.tmp（每个都是同内容副本）。
5. **SillyTavern 多用户账号系统**：用户数据在 `data/_storage/<sha256>` 的 JSON 文件，每个用户一个独立文件。`handle` 可能和 `name` 不同（例：`default-user` 的 name 是 `起早王`）。
