# OpenClaw + DeepSeek 性能对比基准

## 测试环境

| 项目 | Agent A (基线) | Agent B (测试) |
|------|---------------|---------------|
| **平台** | Windsurf Cascade | OpenClaw 2026.4.26 |
| **模型** | Claude Sonnet 4 | DeepSeek V4 Flash / V4 Pro |
| **部署** | 云端 API | 云端 API (api.deepseek.com) |
| **项目** | ForensicAI v0.2.0 | ForensicAI v0.2.0 |

## 对比维度

1. **解题准确率** — 正确 / 部分正确 / 错误
2. **解题速度** — 从提问到最终答案的时间
3. **推理质量** — 证据链完整度、逻辑严密度
4. **成本** — API 费用（按 token 计费）
5. **工具调用** — 能否正确使用取证工具链
6. **格式遵从** — 是否遵守 5 字段答案格式

## 测试题目

建议使用 FIC2026 已解题目作为基准（已知正确答案）：
- C01 (Windows 磁盘 - 基础)
- C05 (Windows 磁盘 - 中等)

- M02 (手机取证 - 中等)
- I03 (综合 - 困难)
