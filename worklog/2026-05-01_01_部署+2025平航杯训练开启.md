1. **winget 源 CDN 抽风**（0x8a15000f）→ 直接走 `install_tools_direct.ps1` 下载官网 installer
2. **aka.ms 间歇性 502** → `aka.ms/wslubuntu2204` 有时连得上有时 502，GitHub raw 基本不通；**能下 GitHub release（github.com + release-assets.githubusercontent.com）**
3. **wsl.exe inbox 版不支持 `--web-download`** → 必须先装 `wsl.2.3.26.0.x64.msi`（GitHub WSL release）升级
4. **0x80370102** → 拯救者 Y9000P 2024 BIOS 默认关虚拟化，F2 进 BIOS → Configuration → Intel Virtualization Technology → Enabled
5. **WSL localhost 代理警告** → 用户 Windows 上有代理软件，WSL 用不了宿主机的 localhost 代理。装 apt / pip 时只要走清华镜像就没事（不走代理）
6. **PowerShell 5.1 读脚本用 GBK** → 脚本不要带中文，或者存 UTF-8+BOM
7. **中文路径 WSL 处理**：`F:\TEXT(important)\Jian cai\2025平航杯\` → WSL `'/mnt/f/TEXT(important)/Jian cai/2025平航杯/'`（单引号避免空格/括号/中文问题）
