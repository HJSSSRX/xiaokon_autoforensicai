# 赛事题型分类（L2）

只记录**考察方向、难度分布、常见坑**。**不写具体答案**。

---

## 使用方式（给 AI）

开工前扫一眼当届比赛的文档，大致知道：
- 会考几种检材
- 常见考点分布（基础信息 / 逆向 / 内存 / 流量 / 域渗透）
- 历届踩过的坑

**不要**把这里的内容当答案用。每一届题目都会变。

| 比赛 | 风格 | 题型主体 | AI 友好度 |
|---|---|---|---|
| 平航杯 | CTF 风，题目多而杂，含大量逆向和脚本分析 | 填空 | 🟢 高 |
| 盘古石杯 | 深度逆向 + Docker 集群 + 数据分析 SQL | 填空 | 🟡 中高 |

| FIC | 规范取证，强调证据链和流程 | 填空 + 简答 | � 中 |

## 共性套路（90% 的赛事都会考）

1. **检材基础信息**：型号/品牌/系统版本/IMEI/时区 → 配置文件 + ALEAPP/EZ 秒出
2. **最近使用痕迹**：最后登录、最近文件、浏览历史 → sqlite + EZ
3. **即时通讯**：微信/QQ 数据库 → 火眼 / PyWxDump
4. **一个恶意样本**：PE 或脚本，配套 5-10 道逆向问题
5. **一段攻击日志 / 流量**：Web 入侵或域横向
6. **加密容器**：VeraCrypt / BitLocker / 加密 zip，密码从其他证据推出来
7. **内存取证**：至少 1 道"从内存拿出密钥/进程/连接"的题
