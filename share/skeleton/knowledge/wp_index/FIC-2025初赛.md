# 2025 FIC 初赛

- **来源**：https://mei-you-qian.github.io/2025/10/28/2025fic初赛复盘/ （玫幽倩）

## 高频套路（自总结）

1. **OBS 录像取证** — 视频本身是检材，里面有时间/IP/密码/工具名（看视频片段+OCR）
2. **音频隐藏线索** — 检材里有.mp3/.wav，需要听完整段落（用 whisper 转文字加速）
3. **手机→电脑钥匙链** — 手机里藏 PC 开机密码、暗号、容器密码（剧情串联）
4. **图片 OCR 取密码** — VeraCrypt 容器密码贴在 1.png 上
5. **OSINT 链** — 域名 → ENS/HNS 区块链 → DNS → github → 旧博客；用 dig/dnstwist/web.archive.org
6. **MD5 反查文件** — 题目给 MD5 让找文件，火眼/EZ 全文 hash 索引
