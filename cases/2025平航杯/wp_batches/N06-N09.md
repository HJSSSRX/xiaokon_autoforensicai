## N06 — 起早王真名

> 起早王的真名是什么（格式：Cai_Xu_Kun 每个首字母均需大写）

**答案**：**未解**
**置信度**：—

### 解析

#### 识别
"真名"应区别于化名"起早王"。格式要求三字拼音（姓_名字_名字）。

#### 已尝试提取方法
1. **BLE device name** — 全部扫遍：`Flipper 123all`, `Cracked`, `QQ_WF_SP8OON`, `LE-YANG QC35 II`, `RAPOO 5.0MS` 等，均为设备型号，非人名。
2. **BLE manufacturer data** 解码：`0x0701434839313430` → `\x07\x01CH9140`（芯片型号）；Xiaomi MiBeacon 结构化数据无明文名字。
3. **BLE scan response**（1348 包）：无新 device name。
4. **BLE 所有可打印字符串（strings -n 6）**：无三字拼音 `Xxx_Xxx_Xxx` 或 `Xxx Xxx Xxx` 模式。
5. **USBPcap device descriptors**：ITE 048d:c102（内置键盘芯片）等厂商 ID，无人名。
6. **USBPcap HID 解码**：倩倩与闺蜜聊天涉及的名字为：
   - `wang qi zhao`（王起早，化名）
   - `qi zao wang`（起早王，同化名倒序）
   两者都是题面已知的化名。

#### 分析
真名大概率在 **其他检材**：
- `window.e01` 里起早王的日记、SillyTavern 角色卡、或 neo4j 数据库
- `20250415_181118.zip` 倩倩手机里的通讯录

本题被归入流量分析板块但实际越界。暂**跳过**，待做 C/M 板块时回填。

## N07 — cmd 命令条数

> 起早王对倩倩的电脑执行了几条 cmd 里的命令（格式：1）

**答案**：`7`
**置信度**：高

### 解析

#### 识别
USB HID 键盘流量解码后，区分："在 cmd 窗口内执行的命令" vs "打开 cmd 前的聊天打字"。

#### 提取
```bash
tshark -r USBPcap -Y 'usb.src=="2.4.1" && usb.data_len==37' \
  -T fields -e frame.time_epoch -e usbhid.data > /tmp/kbd.tsv
```

键盘在 device `2.4`（ITE 048d:c102，联想笔记本内置键盘），HID 报告 37 字节（Report ID + Modifier + Reserved + 6×Key + padding）。

#### 分析
HID 解码后（见 `cases/2025平航杯/artifacts/decode_hid.py`，含 CapsLock 状态机）：

```
[0] bao bao,... qi zao wang ta shuo ta ai wo,... hao fan ar | cmd  ← 聊天+打开 Run
[1]  whoami
[2] net user
[3] net user qianqianwoaini$ abcdefghijkImn /add
[4] net localgroup administrators qianqianwoaini$ /add
[5] net user qianqianwoaini$ /del
[6] net localgroup administrators qianqianwoaini$ /add
[7] rundll32 url.dll,FileProtocolHandler https://fakeupdate.net/win10ue/bsod.html
```

`cmd` 本身是 Win+R 的 Run 输入，非 cmd 内命令。[1]-[7] 为 cmd 内 7 条命令。

#### 验证
```bash
python3 decode_hid.py | grep -c '^\['
# 8 个 line (含 [0] 聊天行 + 7 条命令)
```

- **数据来源**：本地 `USBPcap`
- **工具**：tshark, python3
- **脚本**：`cases/2025平航杯/artifacts/{extract_kbd.sh, decode_hid.py}`

## N08 — 影子账户名 + 密码的 md5

> 倩倩电脑中影子账户的账户名和密码（格式：md5(账号_密码)）

**答案**：`53af9cd5e53e237020bea0932a1cbdaa`
**置信度**：高

### 解析

#### 识别
Windows 影子账户 = 用户名以 `$` 结尾，使 `net user` 命令默认不列出。

#### 分析
从 N07 解码的命令 [3]：
```
net user qianqianwoaini$ abcdefghijkImn /add
```

- **账号**：`qianqianwoaini$`
- **密码**：`abcdefghijkImn`（CapsLock 状态机正确处理：`abcdefghijk` + CAPS ON 打 `i`→`I` + CAPS OFF 打 `mn`）

拼接：`qianqianwoaini$_abcdefghijkImn`

#### 验证
```python
import hashlib
hashlib.md5(b'qianqianwoaini$_abcdefghijkImn').hexdigest()
# 53af9cd5e53e237020bea0932a1cbdaa
```

### 不作弊声明
- **数据来源**：N07 键盘解码
- **工具**：python3 hashlib
- **脚本**：`decode_hid.py` 末尾已打印

## N09 — 最后一条命令的 md5

> 起早王对倩倩的电脑执行的最后一条命令（格式：md5(完整命令)）

**答案**：`0566c1d6dd49db699d422db31fd1be8f`
**置信度**：高

### 解析

#### 识别
命令序列最后一条。

#### 分析
从 N07 命令 [7]：
```
rundll32 url.dll,FileProtocolHandler https://fakeupdate.net/win10ue/bsod.html
```

经 CapsLock 状态机正确解码：`<CAPS>f<CAPS>ile<CAPS>p<CAPS>rotocol<CAPS>h<CAPS>andler` → `FileProtocolHandler`。

此命令在 cmd 里运行 `rundll32.exe url.dll,FileProtocolHandler <URL>`，效果等同于在默认浏览器里打开 https://fakeupdate.net/win10ue/bsod.html（一个假蓝屏恶搞页面）。

#### 验证
```python
import hashlib
hashlib.md5(b'rundll32 url.dll,FileProtocolHandler https://fakeupdate.net/win10ue/bsod.html').hexdigest()
# 0566c1d6dd49db699d422db31fd1be8f
```

### 不作弊声明
- **数据来源**：N07 键盘解码
- **工具**：python3 hashlib
- **脚本**：`decode_hid.py`
